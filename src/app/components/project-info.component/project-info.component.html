Why & How

Made with purpose, not just code.

This project isn't just a demo — it's a full product built to understand the complete lifecycle of a modern Angular web application.

Why It Was Built

The goal was simple: stop reading tutorials and start building something real. A chat app touches every major concept — routing, state, components, real-time UX, responsive layouts — all at once.

Angular from the Ground Up

Every page is a standalone Angular component. TypeScript covers 44% of the codebase — keeping the logic predictable, the bugs fewer, and the developer experience genuinely enjoyable.

A Real Backend

The API was built from scratch in C# and runs on a SQL database — not a mock, not a Firebase shortcut. Real endpoints, real queries, real data persistence.

CSS Written by Hand

No Tailwind. No Bootstrap. Every gradient, animation, card, and layout was written in raw CSS — 35% of the codebase — to master the craft rather than rely on a framework.

Getting Started

Run it in 4 steps.

Clone it, install, serve — and you're live on localhost.

01
Clone the repository
git clone https://github.com/nikanatenaze/Chat-Program-WEB.git
02
Install dependencies
npm install
03
Start the development server
ng serve
04
Open in your browser
http://localhost:4200
